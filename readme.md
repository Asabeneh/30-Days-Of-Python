# 🐍 30 Days Of Python – My Solutions

This repository contains my personal solutions to the exercises from the **30 Days of Python** challenge.

The original project was created by Asabeneh and is available here:  
https://github.com/Asabeneh/30-Days-Of-Python

I forked the repository and am working through each day’s lessons and exercises to strengthen my Python programming skills.


## 📚 About the Challenge

The **30 Days of Python** challenge is a structured learning program designed to help beginners and intermediate programmers improve their Python skills through daily lessons and exercises.

Each day introduces new Python concepts and includes practical tasks to reinforce learning.

Topics covered include:

- Python basics
- Variables and data types
- Conditionals
- Loops
- Functions
- Lists, tuples, sets, and dictionaries
- File handling
- Modules and packages
- Object-oriented programming
- Error handling
- Working with external libraries


## 🎯 Goals

My goals for this repository are to:

- Complete all **30 days of exercises**
- Practice writing clean and readable Python code
- Improve problem-solving skills
- Track my progress publicly
- Build a strong Python foundation

## 📖 How to Use

If you want to try the challenge yourself:

1. Clone the original repository

```bash
git clone https://github.com/Asabeneh/30-Days-Of-Python.git
```

2. Or clone this repository to see my solutions

```bash
git clone https://github.com/shaunmane/30-Days-Of-Python
```

3. Navigate to any day and run the Python files

```bash
python filename.py
```
